Tortilla Mud Client
Клиент для многопользовательских онлайновых текстовых игр в жанре MUD.

Системные требования:
- Windows XP и выше.
- Linux, MacOS (с помощью Wine).

О клиенте можно прочитать тут:
- В справке клиента (вводить в комадной строке клиента):
-- #help
-- #help faq
-- #help triggers
-- #help features
-- #help capabilities
-- #help message

- На сайте программы: https://tmud.github.io/
- В группе Вконтакте: http://vk.com/club92727293

Основные возможности клиента:
- Поддерживаются основные элементы для настройки игры:
- Триггеры (actions)
- Замены (subs) с поддержкой цвета
- Подсветки (highlights), включая подчеркивание, курсив, обрамление
- Фильтры (gags)
- Антизамены (antisubs)
- Макросы (aliases)
- Таймеры (timers)
- Автоподстановки (tabs)
- Группы (groups)

- Кодировки сервера - win и utf8.
- Протоколы MCCP, MSDP, MTTS, OSC palette.
- Работа с двойным IAC (буква я в win кодмровке)
- 256 цветов с палитрой первых 16 цветов.
- Плавающие output-окна.
- Запись логов игры в форматах html или txt.
- Работа с буфером обмена. Вставка из буфера и копирование 
в буфер обмена нужной части текста.
- История мада. Можно читать историю совместно с игрой.
- История последних команд, с автоподстановкой из истории.
- Многофункциональная командная строка.
- Система профилей.

Поддержка плагинов на Lua и С++ (доступны и другие языки при 
необходимости). Уже написано более 20 различных плагинов, 
например такие, как:
- автоперенос длинных строк
- работа со звуком (звуковые эффекты, фоновая музыка)
- синтезатор речи по тексту (говорилка)
- игровая панель для игры мышкой
- дополнительная панель для нужной инфы, с поддержкой тикера
- панель с полосками здоровья, энергии, маны и опыта
- база вещей (лор)
- маршруты (speedwalks)
и многие другие
